## 프로젝트를 마무리 하면서 🥳

코딩 공부를 오르모 부트캠프를 시작하면서 같이 시작을 하고, 모든 언어를 이번에 처음 배우다 보니,  배우는 것에 있어서 다른 동기분들 보다 많이 뒤쳐지는 것도 느꼈고, 그러다보니 프로젝트에 대해 걱정도 많았던 것 같다. 

### 코딩 공부 3주차, 첫 프로젝트 성공

내가 생각했던 것보다는 무사히, 잘 마무리 한 것 같은데, 아쉬움도 많이 남는 것 같다 🙂
